## 2. 디자인 트렌드 (2024~2025)

### 시각적 요소
- **그라데이션 카드**: 단색보다 그라데이션 배경으로 CTA/혜택 강조
- **스텝/티어 시각화**: 2명·5명 같은 단계를 뱃지·카드로 분리
- **금액 강조**: 혜택 금액을 24~32px, 볼드로 눈에 띄게
- **아이콘 활용**: 사람·선물·화살표 등 직관적 아이콘

### UX 원칙
- **한 화면, 한 목적**: 토스처럼 한 화면에 메시지 하나에 집중
- **행동 유도**: "친구 초대하기"·"링크 복사" 버튼이 화면 중앙·하단에 눈에 띄게
- **즉각적 피드백**: 공유·복사 후 "복사됨!" 등 바로 표시
- **진행 상황**: "N명 추천 달성" 진행 바로 동기 부여

### 컬러·톤
- **대리운전 앱에 적합한 톤**: 블루·그린(신뢰, 이동), 골드/옐로우(보상 강조)
- **현재 앱**: `accentBlue`, `accentYellow` 유지 권장

## 3. 화면별 개선 아이디어

### 3-1. 내추천인 등록 (referrer_screen.dart)

**현재**: 흰 카드 + 텍스트 + 번호 입력

**개선 예시**
- 상단에 **히어로 영역**:  
  "친구 초대하고 10,000원 받기" + 큰 금액
- **양방향 혜택 카드**:  
  - 왼쪽: "나" 10,000원  
  - 오른쪽: "추천인" 2,000원
- **입력 필드**: 플레이스홀더·에러 메시지 명확히, 번호 형식 자동 포맷
- **CTA 버튼**: 넓게, 색상 강조, "추천인 등록하고 10,000원 받기" 등 구체적 문구

### 3-2. 쿠폰 화면 추천 배너 (coupon_screen.dart)

**현재**: 초록 그라데이션, 2명/5명 텍스트 나열

**개선 예시**
- **미션 카드**: 각 단계를 카드로 분리
  ```
  [ 2명 추천 ]  스타벅스 쿠폰 2장  ✓달성 / ○미달성
  [ 5명 추천 ]  교촌치킨 세트     ○미달성
  ```
- **진행 바**: "현재 1/5명" 같은 진행률 표시
- **그라데이션**: 더 모던하게 `green-100 → emerald-300` 또는 블루 계열
- **버튼**: "친구 초대하고 보상 받기" 등 행동 유도 문구

### 3-3. 내추천인 현황 (referrer_status_screen.dart)

**현재**: 요약 카드 + 1차/2차 탭 + 테이블

**개선 예시**
- **대시보드 카드**:  
  "N명 추천 완료", "다음 보상까지 M명"
- **업적/배지 느낌**:  
  "2명 달성! 스타벅스 쿠폰"처럼 달성 단계 강조
- **리스트**: 테이블 대신 카드 리스트 (전화번호 마스킹, 날짜, 상태)
- **비어 있을 때**:  
  "아직 추천인이 없어요. 친구를 초대해 보세요!" + CTA 버튼

### 3-4. 홈 화면 친구추천 배너

**현재**: 파란 버튼 "친구 추천하기"

**개선 예시**
- **혜택이 보이는 카드**:  
  "10,000원 + 2,000원"처럼 금액을 배지로
- **공유 버튼**: 카카오톡·문자 등 공유 채널 아이콘
- **미니 진행률**:  
  "2/5명 달성" 같은 짧은 진행 표시

## 5. 일등대리에 바로 적용할 만한 체크리스트

- [ ] 혜택 금액(10,000원, 2,000원)을 더 크고 눈에 띄게
- [ ] "나 / 추천인" 양방향 혜택을 한눈에 보이게
- [ ] 2명·5명 단계를 카드/뱃지로 구분
- [ ] 진행률 바 또는 "N/M명 달성" 텍스트 추가
- [ ] CTA 문구를 "추천인 등록하고 10,000원 받기"처럼 구체적으로
- [ ] 빈 상태 화면에 행동 유도 문구 + 버튼
- [ ] 공유/복사 후 즉각적인 피드백(스낵바, 토스트)

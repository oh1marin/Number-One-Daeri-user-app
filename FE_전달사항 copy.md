# FE 전달사항 — 친구 추천 이벤트 & 마일리지

---

## 마일리지 구조

| 구분 | 금액 |
|------|------|
| 앱 가입 시 | 10,000원 |
| 카드 결제 시 (이용금액) | 5% 적립 |
| 출금 | 잔액 전액 가능 (`POST /withdrawals`) |

---

## 추천 이벤트

| 구분 | 혜택 |
|------|------|
| 추천인 | 친구 등록 시 2,000원 |
| 친구 | 10,000원 |
| 추천인 | 친구 **첫 이용** 시 3,000원 추가 |
| 추천인 | 친구 이용 시 이용금액의 **2%** 적립 |
| 추천인 | **2명** 추천 시 스타벅스 쿠폰 2장 (마일리지 아님) |
| 추천인 | **5명** 추천 시 교촌치킨 세트 (마일리지 아님) |

---

## API 변경 없음

- `POST /referrals/register` — 동일
- `GET /referrals/my` — `tierCoupons` 추가: [{ tier, rewardType, name, earnedAt }]
- `GET /users/me/mileage` — 동일 (balance, withdrawable)
- `GET /mileage/history` — 동일 (description에 적립 사유 표시)

---

## UI 표시 참고

- 가입 시 10,000원 문구
- 추천인 혜택: 2,000원 + 첫 이용 3,000원 + 이용 시 2%
- 2명/5명 보너스 안내
